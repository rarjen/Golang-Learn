Tipe Data String
● String ada tipe data kumpulan karakter
● Jumlah karakter di dalam String bisa nol sampai tidak terhingga
● Tipe data String di Go-Lang direpresentasikan dengan kata kunci string
● Nilai data String di Go-Lang selalu diawali dengan karakter “ (petik dua) dan diakhiri dengan 
karakter “ (petik dua)

Function Data String
● len("string")       -> Menghitung karakter string
● "string"[number]    -> Mengambil karakter pada posisi yg ditentukan
